Introduction

This app uses the r package brainR to display an interactive 3D brain. This app was originally build to create an easy to use environment to display changes in the pattern of brain activations (for simplicity this is not included in this demo app).

Functionality

The functionality of this app includes:

  • Interactive Movement of the brain by simply clicking on it.
  • Changing the transparency values of the displayed 3D brain by using the slider option
  • Changing the color of the displayed 3D brain by specifying a color from the drop-down menu.
  • NOTE: the value set for color and transparency interact with each other.

Changing transparency

Changing color

You can change the color of the brain to a set of predefined colors (yellow, red, blue, white). Note that the transparency and color values interact with each other, thus it is possible to have a brain being colored and transparent at the same time!
